# Archive Map

This archive preserves historically useful exploration while keeping the public entrypoints clean. For key archived scripts with non-obvious names, a one-line top-of-file comment explains their purpose and corresponding conclusion.

- `archive/overnight/`: Early exploration runs that established the phenomenon on state5/7/8 and led to Template A, later refined into Template B.
- `archive/prompt_only/`: Low-signal prompt-only and GCG suffix attack attempts that motivated the shift to the visual channel.
- `archive/tier1_negative/`: Tier1 adaptive-window experiments on state4/6/12 where VIS remained negative, establishing the window-specificity boundary.
